## console.log函数

    在开发中 我们经常会用 console.log 调试我们的代码 
    但是开发移动端的时候 用 chrome的模拟器 和 真机调试 往往差别很大 
    我们需要把开发好的网页放到真机里调试 
    但是这个时候 在真机里 无法观察 console.log 
    如果有 bug 会导致我们无从查起
 - arguments (args)
    arguments 是一个对应于传递给函数的参数的类数组对象。
    可以引用函数的参数
    arguments对象不是一个 Array 。它类似于Array，
    但除了length属性和索引元素之外没有任何Array属性
